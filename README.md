Warden — детектор деаутентификационных атак

Что делает:

1. Мониторит логи роутера в реальном времени — ищет признаки деаут-атак (пакеты отключения клиентов)
2. Обнаруживает атаку — если за 5 секунд находится 5+ подозрительных событий, считает это атакой
3. Шлёт уведомление в Telegram бота — сразу оповещает владельца сети с полной информацией:
   · сколько подозрительных событий
   · сколько атак за сессию
   · сколько клиентов сейчас в сети
   · время атаки
   · рекомендации по защите
4. Не спамит — если атака идёт долго, уведомление приходит не чаще раза в минуту
5. Считает атаки — при повторных атаках увеличивает счётчик, после 3х атак шлёт критическое предупреждение
6. Пишет логи — все события сохраняются, видно когда началась и закончилась атака
7. Работает на любом устройстве — роутер с OpenWRT, Raspberry Pi, обычный компьютер, старенький ноутбук
8. Жрёт 5% CPU — лёгкий, не нагружает систему

Для чего нужен:

Обычный пользователь даже не знает, что его Wi-Fi взламывают.
Wi-Fi Guard — это система раннего предупреждения.
Как только кто-то начал деаут-атаку (первый шаг к взлому пароля), ты получаешь сообщение в Telegram и можешь принять меры: сменить пароль, отключить подозрительные устройства, обновить прошивку.

Установка:

git clone https://github.com/gosduma-opsec/WiFi-monitoring/blob/main/warden.py

pip install requests

python3 warden.py
